📱 Общее описание
App Store Reviews Parser — это веб-приложение на базе Streamlit для сбора, анализа и экспорта отзывов из Apple App Store. Скрипт автоматически фильтрует дубликаты, предоставляет статистику и позволяет удобно работать с собранными данными.

🎯 Назначение
Скрипт предназначен для:
Сбора отзывов пользователей из App Store по ссылке на приложение
Фильтрации дубликатов (API Apple часто возвращает повторяющиеся отзывы)
Анализа тональности и распределения оценок
Экспорта данных в CSV и Excel форматы
Визуализации статистики в реальном времени

✨ Основные возможности
1. Сбор данных
Извлечение ID приложения и страны из ссылки App Store
Парсинг до 1000 отзывов (с учетом ограничений API)
Автоматическая фильтрация дубликатов по ID и содержимому
Адаптивная пауза между запросами для избежания блокировки

2. Аналитика и статистика
Подсчет общего количества уникальных отзывов
Вычисление среднего рейтинга
Распределение оценок (5★, 4★, 3★, 2★, 1★)
Анализ временного периода отзывов
Подсчет положительных, нейтральных и отрицательных отзывов

3. Фильтрация данных
По оценкам (можно выбрать любое количество звезд)
По ключевым словам (поиск в заголовке и тексте)
По временному периоду (выбор диапазона дат)

4. Визуализация
Интерактивные метрики (карточки с ключевыми показателями)
Столбчатая диаграмма распределения оценок
Два режима отображения отзывов:
Компактный (таблица)
Подробный (карточки с форматированием)

5. Экспорт данных
Выгрузка отфильтрованных данных в CSV (UTF-8 с BOM)
Экспорт в Excel с двумя листами:
Отфильтрованные отзывы
Статистика

🛠 Технические характеристики

Используемые технологии
Streamlit — веб-фреймворк для создания интерактивных приложений
Pandas — обработка и анализ данных
Requests — HTTP-запросы к API Apple
OpenPyXL — создание Excel-файлов
Regex — извлечение данных из URL

API источник
Apple iTunes RSS Feed: https://itunes.apple.com/{country}/rss/customerreviews/page={page}/id={app_id}/sortby=mostrecent/json

Ограничения API
Максимум 10 отзывов на страницу
Доступно около 50 страниц (~500 отзывов)
Часто возвращает дубликаты (до 30-40%)
Нет официальной документации по лимитам

🚀 Инструкция по установке и запуску
1. Установка зависимостей
bash
pip install streamlit pandas requests openpyxl
2. Сохранение скрипта
Сохраните код в файл app_store_parser.py
3. Запуск приложения
bash
streamlit run app_store_parser.py
4. Использование
Откроется браузер с веб-интерфейсом
В боковой панели вставьте ссылку на приложение из App Store
Выберите желаемое количество отзывов (100-1000)
Нажмите "Начать сбор"
Дождитесь завершения парсинга
Используйте фильтры для анализа
Скачайте результаты в нужном формате

📊 Примеры использования

Для маркетологов и ASO-специалистов
Анализ тональности отзывов конкурентов

Отслеживание динамики оценок после обновлений

⚠️ Важные замечания
Ограничения API Apple:
Невозможно собрать все отзывы (только последние ~500 уникальных)
API возвращает много дубликатов
Нет возможности получить отзывы старше определенной даты
Этичность использования:
Соблюдайте политику использования Apple
Не делайте слишком частые запросы (в скрипте есть задержки)
Используйте данные только для анализа, не для массовых рассылок
Юридические аспекты:
Отзывы являются публичными данными
При публикации результатов рекомендуется анонимизировать имена пользователей
